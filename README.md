# cloudflare-bypass-botcaptcha-demo

Research/Education purpose

## Prerequisites

```yaml
node.js: 22.22.3
```

## Overview

This repository illustrates how to bypass a Cloudflare Challenge page protected by Turnstile CAPTCHA using Puppeteer in combination with SolveCaptcha.

## Getting Started

```bash
export APIKEY=your_api_key

npm install
npm start
```
